# Dot files directory

### Dependencies

- git: pacman -S git
- GNU stow: pacman -S stow
- neovim: pacman -S neovim


### Usage

Clone directory in home folder

``` git clone https://github.com/MIchael-Kik/dotfiles.git ```

then move to and use GNU stow to create sym-links

```mv ./dotfiles```
```stow .```
